Serial Killer Project

This project uses a comprehensive U.S. homicides dataset to identify potential serial killer activities. Through advanced clustering algorithms, the analysis organizes over 800,000 unsolved cases into distinct groups based on location, victim demographics, and method of killing. The project features an interactive Tableau dashboard that allows for real-time adjustments to cluster sensitivity. The end goal is to assist law enforcement agencies by flagging areas or methods where serial killing might be a plausible explanation for a series of unsolved homicides.
